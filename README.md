## Overview
BlackJackGame is a C# console application that simulates the classic Blackjack card game.  
It includes unit tests with xUnit and uses GitHub Actions for continuous integration.

## Features
- Play Blackjack against the dealer
- Track wins, losses, and draws
- Unit tests with xUnit
- Automated builds and tests via GitHub Actions

## Getting Started

### Clone the repository
```bash
git clone https://github.com/thairangov-oss/BlackJackGame.git
cd BlackJackGame
dotnet build
dotnet run
```

### Running Tests
cd BlackJackGame.Tests
dotnet test
